# 🌱 Система автополива растений с защитой от протечки

Профессиональная система автоматического полива для комнатных растений с функцией защиты от протечек и управлением фитолампой. Проект реализован на платформе Arduino.

## 📋 Описание проекта

Система предназначена для автоматического поддержания оптимальной влажности почвы с помощью помпы. Основные возможности:

- **Автоматический полив** — измерение влажности почвы каждые 30 секунд и включение помпы при сухой почве
- **Защита от протечки** — аварийная остановка полива при обнаружении утечки воды
- **Контроль уровня воды** — блокировка работы помпы при отсутствии воды в резервуаре
- **Управление фитолампой** — ручное включение/выключение дополнительного освещения по кнопке
- **Световая индикация** — красный светодиод сигнализирует об отсутствии воды (постоянное свечение) или протечке (мигание)
- **Мониторинг через Serial** — все события и данные выводятся в монитор последовательного порта

## 📐 Схема подключения

### Цифровые пины

| Пин Arduino | Компонент | Примечание |
|-------------|-----------|------------|
| D2 | Датчик протечки | INPUT_PULLUP, LOW = протечка |
| D4 | Реле фитолампы | OUTPUT, HIGH = включено |
| D6 | Кнопка | INPUT_PULLUP, замыкание на GND |
| D7 | Датчик уровня воды | INPUT, LOW = вода есть |
| D10 | Красный светодиод | OUTPUT, HIGH = горит |
| D11 | Реле помпы | OUTPUT, HIGH = включено |

### Аналоговые пины

| Пин Arduino | Компонент | Примечание |
|-------------|-----------|------------|
| A5 | Датчик влажности почвы | Аналоговый сигнал 0–1023 |

### Подключение питания

- **Реле модули**: VCC → 5V, GND → GND
- **Помпа**: подключается через нормально разомкнутый контакт реле
- **Датчики**: VCC → 5V (или 3.3V для некоторых датчиков), GND → GND
- **Светодиод**: анод → D10, катод → резистор 220 Ом → GND
- **Кнопка**: один контакт → D6, второй → GND

## ⚙️ Логика работы

### Инициализация (`setup()`)

1. Настройка режимов работы всех пинов (входы/выходы)
2. Установка начального состояния выходов (все выключено)
3. Инициализация Serial-соединения на скорости 9600 бод
4. Первичная проверка датчика протечки
5. Вывод сообщения о запуске системы

### Основной цикл (`loop()`)

Цикл выполняет следующие действия в строгой последовательности:

1. **Проверка протечки** (высший приоритет)
   - Опрос датчика протечки
   - При обнаружении — блокировка полива и включение мигающего сигнала

2. **Обработка кнопки**
   - Чтение состояния кнопки с антидребезгом (50 мс)
   - Переключение состояния фитолампы при нажатии

3. **Управление красным светодиодом**
   - При протечке: мигание с интервалом 500 мс
   - При отсутствии воды: постоянное свечение
   - При норме: выключен

4. **Автоматический полив** (только если нет протечки)
   - Измерение влажности почвы каждые 30 секунд
   - Если почва сухая (<300) и вода есть — запуск помпы

5. **Мониторинг активного полива**
   - Проверка наличия воды в реальном времени
   - Контроль времени полива (максимум 5 секунд)
   - Проверка достижения порога влажности


## 📦 Необходимые библиотеки

Проект использует только стандартные библиотеки Arduino, которые уже включены в IDE:

- `<Arduino.h>` — базовая библиотека (подключается автоматически)

Для работы достаточно установить Arduino IDE и выбрать правильную плату.

## 🚀 Быстрый старт

1. Подключите все компоненты согласно схеме выше
2. Откройте скетч в Arduino IDE
3. Выберите вашу плату (**Tools → Board**)
4. Выберите правильный порт (**Tools → Port**)
5. Загрузите прошивку (**Upload**)
6. Откройте **Serial Monitor** (скорость 9600 бод)

### Настройка порогов

В начале скетча находятся параметры для настройки под ваши датчики:

```cpp
const int SOIL_DRY_THRESHOLD   = 300;   // Порог сухой почвы
const int SOIL_WET_THRESHOLD   = 300;   // Порог влажной почвы
const unsigned long MEASURE_INTERVAL    = 30000;   // Интервал измерений (мс)
const unsigned long MAX_PUMP_DURATION   = 5000;    // Макс. время полива (мс)
```

## 📊 Выводимые данные (Serial)

Система выводит в последовательный порт:

- Текущую влажность почвы в процентах и сырое значение (0–1023)
- Статус наличия воды в резервуаре
- События включения/выключения фитолампы
- Сообщения о начале и остановке полива с указанием причины
- Предупреждения о протечках и отсутствии воды

## ⚠️ Меры предосторожности

1. Все электрические соединения должны быть надёжно изолированы
2. Не допускайте попадания воды на плату Arduino
3. Используйте герметичные корпуса для электроники
4. Перед первым запуском проверьте все соединения без подачи воды
5. Рекомендуется использовать отдельный блок питания для помпы
